=== WTC Email Subscription ===
Contributor: Radueff



== Installation ==
1. Copy folder with this plugin to /wp-content/plugins/
2. Go to your admin area and select Plugins
3. Find WTC Contact Form 7 math captcha in the list of plugins and click Activate
4. Go to your admin area and select Contact Form 7, open any form and click Math Captcha tag button
